Technologies
We are equipped with a modern tech stack and skills to execute a project from scratch to completion in a time bound window.

Policybazaar Support Dashboard

The goal of this project was to revamp the My Account dashboard at Policybazaar to provide a 100% unassisted post sales customer experience. The product led to a 10% decrease in support calls, thereby saving 800+ man hours/month for the 10k+ call centre support staff.

Oyo Website and Mobile App

OYO is a hotel booking platform live in 23 countries. The goal of this project to increase conversion at various steps in the user jounrney, while improving the user experience.

BharatPe Mobile App

BharatPe is one stop platform for 5 million Indian MSME's (medium and small enterprises). They offer loans, insurance, e-commerce and daily settlements via QR code to their users. This project involved building their e-commerce marketplace from scratch, solving for customer support and building a B2C credit channel from scratch.

The 13x29 Project

13x29 is a data visualization project that showcases the analysis of reactions to the Supreme Court Verdict to decriminalize homosexuality in India. Tweets from 377 people - 29 people each from 13 different fields were scraped and analysed as part of this study.

Oyo Life App

OYO Life is a urban co-living solution in India, Indonesia and Japan. This project involved designing an app from scratch to streamline operations and create a community.
